🚀 Formação Backend JS

Bem-vindo à Formação Backend JS, uma trilha estruturada para transformar iniciantes em desenvolvedores Backend preparados para o mercado.

Esta formação não é apenas um conjunto de aulas.
Ela é uma jornada prática baseada em desafios reais e progressão técnica, onde o aluno aprende construindo.

🎯 Objetivo da Formação

Capacitar o aluno a:

Criar APIs do zero com Node.js e NestJS
Trabalhar com bancos de dados relacionais e não relacionais
Modelar e estruturar dados de forma eficiente
Construir APIs REST completas
Aplicar boas práticas de arquitetura
Containerizar aplicações com Docker
Orquestrar ambientes com Docker Compose
Implementar autenticação e segurança
Escrever testes automatizados
Construir um backend completo pronto para produção

Ao final, o aluno estará preparado para sua primeira vaga como Desenvolvedor Backend Júnior.

🧱 Estrutura da Formação

A formação é dividida em 3 níveis, organizados em desafios progressivos.

Cada nível representa uma evolução real no pensamento de um desenvolvedor backend.

🟢 Nível 1 — Fundamentos Backend

O aluno sai do zero e constrói suas primeiras bases.

Desafios:

CRUD com MySQL
Estrutura de dados com MongoDB
Primeira API com NestJS
Primeiro container com Docker
Primeiros testes com Jest
🟡 Nível 2 — Integração e Arquitetura

O aluno começa a pensar como backend developer.

Desafios:

Sistema com relacionamentos (MySQL)
Modelagem de dados em MongoDB (rede social)
API REST completa com NestJS
Ambiente com Docker Compose
Testes com mocks
🔴 Nível 3 — Backend Profissional

O aluno constrói sistemas reais prontos para o mercado.

Desafios:

API estilo Spotify (CRUD + paginação + upload)
Sistema de autenticação completo
Docker multi-stage e otimização
Arquitetura completa com múltiplos serviços
Projeto final completo
