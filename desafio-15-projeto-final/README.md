# 🚀 Desafio 15 — Projeto Final

Este é o desafio de consolidação da Formação Backend JS.

Aqui você irá desenhar, implementar, testar e documentar um sistema completo com escopo mais próximo de mercado.

## 🎯 Objetivo do Desafio

Entregar uma aplicação backend autoral que reúna os principais pilares da formação:

- API bem estruturada;
- persistência SQL e/ou NoSQL;
- autenticação;
- testes;
- execução com Docker.

## 🧠 Tecnologias que você irá praticar

- Node.js e TypeScript
- NestJS ou framework escolhido
- MySQL e/ou MongoDB
- Docker e Docker Compose
- Jest (com unitários e mocks)
- Boas práticas de arquitetura

## 📦 Escopo funcional (MVP)

Defina um domínio (ex.: marketplace, financeiro, cursos, biblioteca) com:

- no mínimo 4 entidades principais;
- autenticação com perfis;
- CRUD completo para ao menos 2 entidades;
- pelo menos 1 fluxo de negócio composto (ex.: checkout, matrícula, aprovação).

## 🛠 Requisitos técnicos

- Arquitetura por camadas ou módulos.
- Validação de entrada em todas as rotas mutáveis.
- Tratamento centralizado de erros.
- Banco versionado com migrações (quando aplicável).
- Ambiente completo via Docker Compose.
- Testes cobrindo regras críticas.

## ✅ Critérios de aceite

- Projeto roda localmente com documentação clara.
- Endpoints principais funcionam de ponta a ponta.
- Segurança básica aplicada (auth + autorização).
- Testes executam sem dependências manuais complexas.

## 📄 Entregáveis

- Código-fonte do projeto.
- `README.md` com:
  - visão geral;
  - arquitetura;
  - instruções de setup;
  - coleção de rotas;
  - decisões técnicas.
- Script de seed (opcional, recomendado).

## ⭐ Parte opcional

- Deploy em ambiente cloud.
- Observabilidade (logs estruturados, healthcheck, métricas básicas).

## 🏁 Resultado esperado

Ao concluir este desafio, você terá um projeto backend completo de portfólio, demonstrando domínio técnico e maturidade profissional.

## 🚀 Próximo nível

A partir daqui, evolua o projeto final de forma contínua: performance, segurança avançada, mensageria e escalabilidade.
